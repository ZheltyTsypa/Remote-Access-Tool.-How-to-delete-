# Что такое RAT?
### RAT (Remote Access Tool) – вредоносное ПО, которое позволяет управлять вашим устройством на расстоянии. В основном RAT позволяет мониторить все ваши действия, читать ваши сообщения а также получить полный доступ к вашим файлам. 
### Первый RAT появился в 1980 году, и назывался он pcAnywhere. Он был полностью легитимым, считался софтом для админов.
### С 1985 – 1990 появились первые трояны-шпионы. Именно тогда ввели термин RAT для вредоносных программ.
- · 1995 год — Back Orifice (шутка хакеров, но показала опасность).
- · 1996 год — NetBus, первый массовый инструмент для слежки.
- · 1998 год — SubSeven, который стал культовым и задал стандарты для современных RAT.
- · 2000-е (расцвет): RAT стали коммерческими продуктами для кибершпионажа (например, PoisonIvy в 2005-м).
- · Сейчас: Используют AI-обходы, маскировку под легитимный трафик и модульные архитектуры.
